如何開始編輯報告書

下載：https://drive.google.com/file/d/1lH3J-EETOpNruAz86Mi8A4PcKkgJOsh2/view

解壓縮完裡面有兩個資料夾

 pandoc-2.4-windows-x86_64 和 miktex_portable

用SciTE 開啟start.bat

並在path=的最後方新增

1
%Disk%:\pandoc-2.4-windows-x86_64;%Disk%:\miktex_portable\texmfs\install\miktex\bin;

接著在cmd(小黑窗)中輸入

1
python -m pip install pandoc-fignos pandoc-tablenos pandoc-eqnos

安裝完成後重新啟動可攜系統即可

參考內容：https://mdecp2018.github.io/finalproject-bgx/content/Kmol-editor.html

參考作法：https://mdekmol.github.io/cd2019a-task1-2019cda_t1_g3_1/content/%E5%A6%82%E4%BD%95%E9%96%8B%E5%A7%8B%E7%B7%A8%E8%BC%AF%E5%A0%B1%E5%91%8A%E6%9B%B8.html

各組完成 html 與 pdf 格式之手足球專案結案報告 (文字結案報告書)
